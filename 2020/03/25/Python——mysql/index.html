    cursor.execute("SELECT VERSION()") # 使用execute方法执行SQL语句

TIPS

  • 1、数据库连接失败
    # TypeError: an integer is required (got type str)
# 原因： mysql默认连接3306端口，如果是其他端口需要使用port字段
# port是数字，不是字符串，不要使用”“
db = MySQLdb.connect("localhost", "py_user", "py_user" ,"py_test", charset="utf8",port=3311)
  • 2、插入数据报错 单引号转义问题
    检查mysql语句，insert into tables(name,pass) values (%s,%s)
